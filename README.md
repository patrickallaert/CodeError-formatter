CodeError-formatter
===================

Pretty print the content of CodeError.js script as generated by projects like
[HipHop for PHP][hiphop].

Requirements
------------

[PHP][php] >= 5.2

Running
-------

CodeErrorFormatter.php /full/path/to/CodeError.js

[hiphop]: https://github.com/facebook/hiphop-php/
[php]: http://php.net/
